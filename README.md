**Project Title**
# Quantum Circuit Simulation using Qiskit

**Short Description**
This project demonstrates core quantum computing concepts using IBM Qiskit, including Bell State entanglement and Quantum Random Number Generation (QRNG).

**Features:**
- Bell State Quantum Circuit
- Quantum Random Number Generator
- Superposition demonstration
- Entanglement simulation
- Quantum measurement output

  **Technologies:**
  - Python
- IBM Qiskit
- Jupyter Notebook

**Installation**
pip install -r requirements.txt

**Run the Project**
python bell_state.py
python qrng.py
